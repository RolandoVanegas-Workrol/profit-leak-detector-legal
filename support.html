Producto: Profit Leak Detector (Shopify app)
Contacto: info@workrol.com
Última actualización: 2026-04-26

Cómo contactarnos

Escríbenos a info@workrol.com indicando: tu tienda (myshopify.com) captura del problema pasos para reproducir.

Conexión con Google Search Console

  • La App usa OAuth 2.0 para conectarse a tu cuenta de Google y consultar métricas reales de Search Console.
  • Si tu proyecto OAuth está en “Testing”, solo podrán autorizar usuarios de prueba (test users). Para uso con clientes, debe estar “In production”.
  • Luego de conectar, selecciona la propiedad verificada (https://... o sc-domain:...) dentro de la App.

Conexión con Google Analytics 4

  • En el Dashboard o en Indexing, haz clic en Conectar GA4.
  • La App solicita acceso de solo lectura a Google Analytics para leer métricas agregadas.
  • Luego de autorizar, selecciona la propiedad GA4 correspondiente a tu tienda.
  • Profit Leak Detector usa esos datos para mostrar sesiones orgánicas, conversiones/eventos clave, revenue y páginas de destino en el dashboard.

Desconectar Google

Puedes desconectar Google Search Console o GA4 dentro de la App. También puedes revocar el acceso directamente desde myaccount.google.com/permissions.

Desinstalación

Puedes desinstalar Profit Leak Detector desde Shopify Admin → Apps. Al desinstalar, la App deja de operar para tu tienda.

Eliminación de datos

Si deseas solicitar eliminación de datos (incluyendo tokens OAuth asociados a tu tienda), escribe a info@workrol.com con el asunto Data Deletion Request e incluye tu dominio *.myshopify.com.

Buenas prácticas

  • Usa periodos consistentes (por ejemplo 28 días) para comparar tendencias.
  • Si un periodo no tiene datos disponibles en Search Console, verás “Sin datos disponibles para este período”.
  • Si GA4 no tiene sesiones orgánicas suficientes en el período seleccionado, el dashboard mostrará “Datos insuficientes” en lugar de inventar conversiones o revenue.
  • Activa el App Embed en tu tema si quieres habilitar funcionalidades relacionadas con Online Store.
